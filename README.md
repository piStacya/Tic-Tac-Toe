# Tic-Tac-Toe

The well-known game “Tic-Tac-Toe” is ideal for passing time during long lectures or taking a break from studying. It allows players to test their logical thinking, apply strategic tactics, and improve problem-solving skills in a simple and entertaining way.

Our version is written in Python, offering a fun way to challenge friends and sharpen strategic skills that are valuable in everyday life—without wasting time mindlessly.
